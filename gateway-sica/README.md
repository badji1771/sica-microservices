# GATEWAY SICA

Cette application joue à la fois le rôle de front end et de gateway 

## Fonctionnalités

- Ouvrir journée
- Fermer journée 
- Lister les journées
- Lister les opérations d'une journée
- Lister les règlements d'une journée
